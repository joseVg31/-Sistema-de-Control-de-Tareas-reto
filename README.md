# 📝 Sistema de Control de Tareas

Una aplicación de escritorio sencilla y funcional construida en **Java** utilizando la biblioteca **Swing**. Este proyecto implementa el patrón de diseño **MVC (Modelo-Vista-Controlador)** para gestionar tareas diarias de forma eficiente.

## 🚀 Características

* **Agregar Tareas:** Permite ingresar el nombre de una tarea y añadirla a la lista.
* **Estado de Tareas:** Visualización clara de si una tarea está "Pendiente" o "Completada".
* **Marcar como Completada:** Cambia el estado de una tarea seleccionada con un solo clic.
* **Eliminar Tareas:** Limpia tu lista eliminando las tareas que ya no necesites.
* **Contador en Tiempo Real:** Un indicador dinámico que muestra el total de tareas actuales.
* **Interfaz Nativa:** Adaptada al sistema operativo para una apariencia moderna.

## 🛠️ Tecnologías Utilizadas

* **Lenguaje:** Java 17+
* **Interfaz Gráfica:** Java Swing / AWT.
* **Arquitectura:** Modelo-Vista-Controlador (MVC).
* **Gestión de Módulos:** Java Platform Module System (JPMS).

## 📂 Estructura del Proyecto

El proyecto está organizado de la siguiente manera:

* `src/main/java/org/example/sistemadecontroldetareas`
    * `Main.java`: Punto de entrada de la aplicación.
    * `model/`: Contiene la lógica de datos (`Tarea.java`).
    * `view/`: Define la interfaz gráfica (`VentanaPrincipal.java`).
    * `controller/`: Maneja la interacción entre la vista y los datos (`TareaController.java`).
